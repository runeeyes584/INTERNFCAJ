### Week 1 Achievements:

- **AWS Fundamentals & Account Administration**: Learned Cloud Computing concepts, core service groups (Compute, Storage, Networking, Security), and AWS Global Infrastructure (Region, AZ, Edge Location). Completed secure account setup (MFA enabled, IAM Admin User, Budget Alerts, Support Plan) and learned cost optimization principles.

- **VPC Network Design & Deployment**: Learned Amazon VPC components (CIDR Block, Public/Private Subnet, Route Table, IGW, NAT Gateway, Security Group, NACL) and practiced building a basic VPC environment connected to an EC2 Instance.

- **Advanced Topics & Troubleshooting**: Studied Hybrid DNS with Route 53 Resolver (Inbound/Outbound Endpoints, Forwarding Rules, On-premises DNS resolution), practiced VPC Peering, and learned to troubleshoot common issues (Route Table, SG, NACL, NAT Gateway, DNS resolution).

- **Learning Progress**: Completed watching and studying videos 1 to 64 in the *First Cloud Journey Bootcamp 2025* series by AWS Study Group.
